# Water Bottle Tracker

This is a small Vite + React + Tailwind demo that hosts the Water Bottle Tracker component from the repo.

Quickstart:

1. Clone:
   - git clone https://github.com/zainamir07654-ship-it/watertracker2.git
2. Install:
   - npm install
3. Run dev server:
   - npm run dev

Open the local dev server URL (usually http://localhost:5173) to see the app.
